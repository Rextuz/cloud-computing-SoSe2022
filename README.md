## Technologies used:
 - Azure CLI
 - Python fabric

Deployment code is in _fabric.py_

## Prerequisites:

Change Azure subscription id in fabric.py

## To deploy the app:

```
fab deploy_all
```

## To upscale:

```
fab upscale
```

## To downscale:

```
fab downscale
```
